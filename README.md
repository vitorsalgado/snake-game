# SnakeJS - Snake Game

Simple Snake Game in JavaScript.

## Game

Available [here](https://vitorsalgado.github.io/snake-game/).

## Running

To run a local server, execute the command:

```
npm start
```

The server will start on [http://localhost:3000](http://localhost:3000).
